Web Development

In a nutshell, web development is using technologies and resources to craft websites. Some of the tools web developers use include HTML for structuring websites, CSS for adding styles to websites, and JavaScript for adding functionality to websites (like events that happen after you click a button!). Web development is a combination of design, code, problem-solving, and creativity!

Responsive Web Design


Abbreviated to RWD, Responsive Web Design is a practice that is involved with building websites in a way that responds to changes in devices (computer, tablets, phones, etc.) so that the content is organized and displayed in the most effective way for that device. It's important to keep RWD in mind when developing websites because it makes it more usable for your users. NOTE: You can use F12 in your browser to check responsiveness!

User Interface Design


Abbreviated to UI Design, User Interface Design is a practive that is involved with designing aesthetically-pleasing pages and graphics. To achieve a successful UI design for your websites, use the elements of art (line, shape, color, value, form, texture, and space). While using these elements, keep in mind the principles of art: balance, contrast, emphasis, movement, pattern, rhythem, and unity/variety.

User Experience Design


Abbreviated to UX Design, User Experience Design is a practice that is involved with laying out and adjusting web pages and their elements in a way that is optimal for the user visiting the page. This includes thinking about what words to use, what labels to have, icons vs text, placement, ordering, spacing, and more! Having a good UX design will make users enjoy the site more and make it easier for them to use the site.

What is Bootstrap?


Bootstrap is a tool for styling and organizing web pages. It provides premade stylesheets with rules that we can use through adding CSS classes to HTML elements to format them. To use these, we need to import Bootstrap into our project in the head tag.

One of the big benefits of using Bootstrap is the grid layout functionality. According to W3Schools, Bootstrap's grid system allows up to 12 columns across the page. If you do not want to use all 12 column individually, you can group the columns together to create wider columns. Bootstrap's grid system is responsive, and the columns will re-arrange depending on the screen size: on a big screen it might look better with the content organized in three columns, but on a small screen it would be better if the content items were stacked on top of each other.

CSS

Cascading Style Sheets, describes the style (color, position, font, etc.) HTML elements chosen by a selector should have based on a defined rule

Element Selectors


When we work with CSS, there are a few different ways we can select HTML elements that we want to style.

HTML Tag Name
To use this as a selector, we need to use the name of the HTML tag for the element (NOT including the brackets!). Some examples would be p, body, h1, img, a, and div.

Class Attribute
To use this as a selector, we need to add a class attribute to the HTML element's tag, such as <img class="nameOfClass">. We use class attributes when we want to use the same style for more than one HTML element.

ID Attribute
To use this as a selector, we need to add an ID attribute to the HTML element's tag, such as <img id="nameOfID">. We use ID attributes when we want to use a style for only ONE HTML element.
